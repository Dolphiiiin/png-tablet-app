JSON API Documentation

このGitHub Pagesサイトは、JSONデータを返すシンプルなAPIエンドポイントとして機能します。

利用可能なエンドポイント

全記事の取得

エンドポイント: /data/articles.json

メソッド: GET

説明: 全ての記事データをJSON形式で返します

レスポンスの例:

[
  {
    "id": "article1",
    "title": "記事タイトル",
    "content": "記事本文...",
    "imageUrl": "画像URL",
    "redirectUrl": "リダイレクト先URL"
  },
  ...
]

個別記事の取得

エンドポイント: /data/article1.json, /data/article2.json, etc.

メソッド: GET

説明: 指定したIDの記事データをJSON形式で返します

レスポンスの例:

{
  "id": "article1",
  "title": "記事タイトル",
  "content": "記事本文...",
  "imageUrl": "画像URL",
  "redirectUrl": "リダイレクト先URL"
}

リダイレクト機能

各記事には対応するリダイレクトURLがあります：

  • /redirect/article1.html → 記事1のリダイレクト先
  • /redirect/article2.html → 記事2のリダイレクト先
  • /redirect/article3.html → 記事3のリダイレクト先
  • /redirect/article4.html → 記事4のリダイレクト先
  • /redirect/article5.html → 記事5のリダイレクト先
